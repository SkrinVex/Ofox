# Обновление чатов - Changelog

## Что исправлено

### 1. Конфликты при смене аккаунта
- **Добавлено поле `userId`** в таблицу `Chat` - теперь каждый чат привязан к конкретному пользователю
- **Добавлено поле `senderId`** в таблицу `Message` - отслеживание отправителя каждого сообщения
- Это предотвращает конфликты при переключении между аккаунтами

### 2. Отображение бейджей в чатах
- **Добавлено поле `userBadges`** в таблицу `Chat` - хранит бейджи собеседника в JSON формате
- Бейджи теперь отображаются рядом с именем пользователя в списке чатов
- Используется компонент `UserBadges` для единообразного отображения

### 3. Относительное время
- Создан общий утилитный файл `TimeUtils.kt` с функцией `formatTime()`
- Время теперь отображается относительно ("только что", "5 мин назад", "2 ч назад", "3 дн назад")
- Применено в:
  - Списке чатов (ChatsScreen)
  - Сообщениях в чате (ChatDetailScreen)
  - Постах (HomeScreen)

## Технические детали

### Формат хранения бейджей
```json
[
  {
    "badge_type": "verified",
    "description": "Верифицированный пользователь"
  },
  {
    "badge_type": "donor",
    "description": "Поддержал проект"
  }
]
```

### Логика относительного времени
- < 5 сек → "только что"
- < 60 сек → "X сек назад"
- < 60 мин → "X мин назад"
- < 24 ч → "X ч назад"
- ≥ 24 ч → "X дн назад"

## Миграция БД
База данных автоматически пересоздаётся при обновлении (используется `fallbackToDestructiveMigration()`).
**Внимание**: все локальные данные будут удалены при первом запуске.
